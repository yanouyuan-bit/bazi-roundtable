# 命理圆桌会 · bazi-roundtable

一个 Claude skill：让**多位不同流派的命理师**对同一张八字命盘进行圆桌论命——多视角总评、交叉质疑、可证伪断语、逐轮记分复盘。

> A Claude skill that stages a multi-school BaZi (Four Pillars) roundtable: fortune-tellers from rival schools analyze the same chart, cross-examine each other's methodology, issue falsifiable predictions with probabilities and time windows, and keep score round by round. Content is in Chinese, matching its use case.

## 为什么做这个

单一 AI 论命有系统性通病：

- 普遍**高估命主层次**，怎么批都是"中上之命"
- **顺着用户说**，用户报出实况后立刻"其实盘里早写了"（事后倒推）
- 喜忌含糊到"喜金水"就完事，**无法准确到字**
- 断语留两个出口，怎么都对，**不可证伪**

这些毛病靠"提醒模型注意"治不好。本 skill 的思路是**结构性对冲**：

| 机制 | 对冲什么 |
|---|---|
| 四个方法论真实分歧的流派互相 counter-review | 单一视角的盲区与趋同 |
| 主持人从严记分：宽泛命中不邀功、含糊断语不入表 | 不可证伪的"怎么都对" |
| 断语强制格式：结论 + 概率 + 应期 + "若……则记我错" | 双头下注与断语漂移 |
| 复盘环节明令禁止倒推，违规点名（六类违规行为清单） | 事后合理化 |
| 场外观众按断语类型重排战绩、检查集体偏差 | 四家共识其实是集体抄社会先验 |

定位是"君子玩易"：把命理当作严肃的解释学与推理训练场，不当安慰工具，也不当宿命剧本——断语当风险清单用，桨在命主自己手里。

## 阵容

- **盲派**：做功、宾主，术语密集，敢下凶断
- **格局派**：子平格局法，引经据典，只答骨架类问题
- **调候派**：《穷通宝鉴》一系，寒暖燥湿，爱押体感类赌注
- **民国派**：韦千里一系，白话直断，敢报具体数字，输得起
- **主持人**：出题、控场、记分、点名违规
- **场外观众**（可选）：外行元批评者

角色被批评时有两种回应档位：**理想版**（认错干脆、立军令状）与**真实版**（护招牌、抬师承、卸责——符合真实从业者利益结构的话术），可对照呈现。

## 安装

### Claude Code

```bash
# 个人级：复制到 ~/.claude/skills/
cp -r bazi-roundtable ~/.claude/skills/

# 或项目级：复制到项目的 .claude/skills/
cp -r bazi-roundtable your-project/.claude/skills/
```

### claude.ai

将 `bazi-roundtable/` 目录打包上传为 skill（Settings → Capabilities → Skills）。

## 使用

贴出八字命盘（四柱干支 + 性别 + 大运），然后：

```
请用圆桌形式批这个盘
乾造：甲子 丙寅 戊午 壬戌，3 岁起运……
```

流程**互动式逐环节推进**：每次只走一个环节，由主持人开场、收场、报下一步选项，然后停下等你下令——你看完这一轮，再说"下一个"。环节顺序：起盘总评 → 交叉 review → 圆桌问答 → 可证伪硬断轮 → 结账复盘（报实况对账记分）→ 前瞻预判 → 观众点评 → 第二次批命，可独立触发、自由组合。中途可随时点名追问某一家、质疑某条断语。也支持轻量模式（"盲派看一眼就行"），或明确说"一次跑完"取消逐轮停顿。

> 逐环节停顿不只是阅读节奏——结账复盘的含金量取决于你在硬断轮之前没有透露实况。别急着报答案，让他们先把赌注挂满。

## 目录结构

```
bazi-roundtable/
├── SKILL.md              # 流程骨架、角色简表、纪律清单、输出规范
└── references/
    ├── discipline.md     # 反倒推纪律、记分规则、真实版话术库、观众框架
    ├── mangpai.md        # 盲派：做功、宾主、作用优先级、取象
    ├── geju.md           # 格局派：成格破格、用神清浊、十神生克
    ├── tiaohou.md        # 调候派：十干十二月调候用神表、寒暖取象
    └── minguo.md         # 民国派：韦千里式批命体例、报数规则
```

每个流派文件末尾附**争议点立场表**（辰土算不算根、合绊损耗几成、虚透之火如何断……），同一争议各派立场互斥——这是圆桌上真实冲突的燃料。

## 边界

- 未成年人感情/关系类问题不提供命理框架分析
- 健康断语止于"提醒体检方向"，不做诊断
- 财务/法律相关断语不构成投资或法律建议
- 断语的实用价值不依赖命理为真——它值钱是因为它敢被证伪
