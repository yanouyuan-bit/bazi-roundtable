# 争议矩阵（交叉 review 弹药库）

本文件是四派方法论分歧的唯一权威来源，**交叉 review 环节前必读**。每个争议至少有两派互斥立场；角色在交叉 review 中提出的"实质分歧"应优先取材于此，也允许基于本矩阵的逻辑延伸出新分歧。

使用规则：

- 立场是**方法论承诺**，不是辩论姿态——角色被追问时必须能用本派框架自洽地守住（守不住就按认错规则处理）
- 同一争议被实况裁决后（如合绊之字在应期年是否起作用），主持人在会议纪要"方法论看点"中记录裁决偏向哪派
- 某派标注"无立场"时必须给出理由，且不得在场上临时编造立场混分

## 1. 辰土算不算木之根

| 派 | 立场 |
|---|---|
| 盲派 | 不算根。辰是水库湿土，乙木在辰只算余气微根；做功中辰的身份看它跟谁作用（会水局则纯做水库论） |
| 格局派 | 按司令与会合论：辰中乙木为余气，木在辰有微根可受生扶；辰戌丑未皆杂气，透出何干用何气。盲派"不算根"过苛，抹掉杂气格一整类理路 |
| 调候派 | 不谈算不算根，谈"辰能养木"：辰是湿土、滋木之土，春木见辰是沃土——根不根与养不养是两个问题，两派都在"根"这个概念里打转 |
| 民国派 | 按余气计三分力，不搞有无之争：辰中乙木什么时候起用，看大运流年是否引出（透乙、会木局之年）——这是可验证的应期抓手，逐盘验 |

## 2. 合绊损耗几成

| 派 | 立场 |
|---|---|
| 盲派 | 看死不看折扣：被合住的字功能基本冻结（七八成以上），除非逢冲解合 |
| 格局派 | 合而不化者减力约半，性质仍在；关键看被合之字是相神还是忌神——相神被合即以破格论，不谈折扣。盲派"冻结论"把量的问题偷换成有无问题 |
| 调候派 | 减力三四成，但**寒暖之性不灭**：丙被辛合仍有暖意，只是不能做功。性与力要分开记账 |
| 民国派 | 无统一折扣（声明的无立场）：合绊损耗因合化条件而异，逐盘验，拒绝先验定数。没立场好过假精确 |

## 3. 虚透之火（无根之神）如何断

| 派 | 立场 |
|---|---|
| 盲派 | 以"象"论不以"力"论——可断名气、门面、虚荣，不可断实质事业；一见水克即为灾 |
| 格局派 | 虚透之神可为格所用（如虚官虚印仍可成格），唯层次降等；"虚透即归零"会错杀大量清贵之格 |
| 调候派 | 调候之火虚透仍有效——气候要的是温度不是力量，一点丙火虚透也解得了金寒水冷之局的三分寒 |
| 民国派 | 打七折计力，不归零：虚透之神在流年得根时兑现，这正是好用的"期权"——断应期的抓手 |

## 4. 旺衰要不要先定

| 派 | 立场 |
|---|---|
| 盲派 | 不需要精确定旺衰：作用关系定了，旺衰自然在里面。旺衰只是作用的"力量系数" |
| 格局派 | 要。身强身弱是取用的前提之一（身弱财格须帮身），跳过旺衰直接谈作用是无根之谈 |
| 调候派 | 先定寒暖再谈旺衰：气候两轴独立于旺衰之外，寒木不论多旺都要火——旺衰是格局家的家务事，答不了体感问题 |
| 民国派 | 要，旺衰是本派四件套之一；但旺衰判断必须接受验证校准——身强身弱之争吵不出结果时，各按己判下注，让实况裁决 |

## 5. 空亡等神煞用不用

| 派 | 立场 |
|---|---|
| 盲派 | 重视空亡：宫位逢空减力，填实、冲空之年为应期 |
| 格局派 | 神煞不入成败判断：格局成败只论十神生克救应，神煞至多做取象点缀，不得改变格的层次结论 |
| 调候派 | 不用：气候系统里没有神煞的位置，寒暖燥湿四字足矣 |
| 民国派 | 冗余，弃用。凡神煞能断的，十神旺衰都能断；不能的，神煞也是碰运气 |

## 6. 调候优先还是格局优先

| 派 | 立场 |
|---|---|
| 盲派 | 都不优先——作用关系优先。寒暖只是取象的一路材料，进不了做功判断的主链 |
| 格局派 | 格局为体、调候为辅：调候是加减分项，不能反客为主推翻成格判断 |
| 调候派 | 失度严重时调候第一：冬金无火，格局再清也是寒儒。失度不严重时自认辅助 |
| 民国派 | 谁可证伪听谁的：失度之盘先验体感断语（怕冷怕热、季节节律），命中则该盘调候权重上调——用战绩分配话语权，不用教义 |

## 7. 层次判断怎么锚定

| 派 | 立场 |
|---|---|
| 盲派 | 做功效率给"格局大小"排序；换算成社会量级误差大，属本派低置信区 |
| 格局派 | 四阶：格真且清（上）、格真带浊（中上）、破格有救（中）、破格无救（下）；须给区间并写明扣分项 |
| 调候派 | 调候得失能断"舒不舒展"，断不了"多大规模"——只对层次做顺逆修正，不做主判 |
| 民国派 | 反对"理路漂亮=层次高"：层次要用可验证代理指标锚定（学历段、行业收入带、职级），宁可报窄区间被打脸，不报宽区间混分 |

## 8. 细节题（形貌、身高、月份应期）该不该答

| 派 | 立场 |
|---|---|
| 盲派 | 有做功可依则敢答敢断（含凶断）；无明显做功的平盘主动声明分辨率不足，拒硬断 |
| 格局派 | 拒答或标注低置信度："此非格局法分辨率所及。"应期粒度到运程段落，硬给月份属超售 |
| 调候派 | 只答体感类细节（寒热偏性、季节节律、气色），六亲具体事件不押——气候是背景板不是扳机 |
| 民国派 | 该答。"细节超出分辨率"成立，但要用拒答换公信力，就必须同时公布自己答了的题的命中率——只拒答不晒战绩是耍赖 |

## 9. 燥湿是否独立于寒暖

| 派 | 立场 |
|---|---|
| 调候派 | 独立两轴：火炎土燥之局补水是调湿不只是降温，癸与壬的取舍由此分（癸为雨露、壬为江河） |
| 盲派 / 格局派 | 不设气候轴：燥湿之性并入作用关系（盲派）或格局配置（格局派）里消化，不单独记账 |
| 民国派 | 可并轴简化：删繁就简，寒暖燥湿并一轴"舒适度"够用；两轴分账若不能带来更准的断语，就是多余精度 |
